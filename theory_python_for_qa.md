## –£—Å–ª–æ–≤–Ω—ã–µ –æ–ø–µ—Ä–∞—Ç–æ—Ä—ã

–û—Å–Ω–æ–≤–Ω—ã–µ —É—Å–ª–æ–≤–Ω—ã–µ –æ–ø–µ—Ä–∞—Ç–æ—Ä—ã

1. if - –∫–ª—é—á–µ–≤–æ–µ —Å–ª–æ–≤–æ, –∫–æ—Ç–æ—Ä–æ–µ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –¥–ª—è –ø—Ä–æ–≤–µ—Ä–∫–∏ —É—Å–ª–æ–≤–∏—è. –ï—Å–ª–∏ —É—Å–ª–æ–≤–∏–µ –∏—Å—Ç–∏–Ω–Ω–æ (True), –≤—ã–ø–æ–ª–Ω—è–µ—Ç—Å—è –±–ª–æ–∫ –∫–æ–¥–∞ –ø–æ–¥ –Ω–∏–º.

3. else - –æ–ø—Ä–µ–¥–µ–ª—è–µ—Ç –±–ª–æ–∫ –∫–æ–¥–∞, –∫–æ—Ç–æ—Ä—ã–π –≤—ã–ø–æ–ª–Ω—è–µ—Ç—Å—è, –µ—Å–ª–∏ –Ω–∏ –æ–¥–Ω–æ –∏–∑ –ø—Ä–µ–¥—ã–¥—É—â–∏—Ö —É—Å–ª–æ–≤–∏–π –Ω–µ –±—ã–ª–æ –∏—Å—Ç–∏–Ω–Ω—ã–º.

```python
    # –∫–æ–¥ –≤—ã–ø–æ–ª–Ω—è–µ—Ç—Å—è, –µ—Å–ª–∏ —É—Å–ª–æ–≤–∏–µ –∏—Å—Ç–∏–Ω–Ω–æ

    # –∫–æ–¥ –≤—ã–ø–æ–ª–Ω—è–µ—Ç—Å—è, –µ—Å–ª–∏ –¥—Ä—É–≥–æ–µ_—É—Å–ª–æ–≤–∏–µ –∏—Å—Ç–∏–Ω–Ω–æ

    # –∫–æ–¥ –≤—ã–ø–æ–ª–Ω—è–µ—Ç—Å—è, –µ—Å–ª–∏ –Ω–∏ –æ–¥–Ω–æ –∏–∑ —É—Å–ª–æ–≤–∏–π –Ω–µ –∏—Å—Ç–∏–Ω–∞
```

–ü—Ä–∏–º–µ—Ä –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è

–ü—Ä–∏–º–µ—Ä: –ø—Ä–æ–≥—Ä–∞–º–º–∞, –æ–ø—Ä–µ–¥–µ–ª—è—é—â–∞—è, —è–≤–ª—è–µ—Ç—Å—è –ª–∏ —á–∏—Å–ª–æ –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω—ã–º, –æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω—ã–º –∏–ª–∏ –Ω—É–ª—ë–º.

- and - –∏—Å—Ç–∏–Ω–∞ —Ç–æ–ª—å–∫–æ —Ç–æ–≥–¥–∞, –∫–æ–≥–¥–∞ –æ–±–∞ —É—Å–ª–æ–≤–∏—è –∏—Å—Ç–∏–Ω–Ω—ã.

- not - –∏–Ω–≤–µ—Ä—Ç–∏—Ä—É–µ—Ç –∑–Ω–∞—á–µ–Ω–∏–µ —É—Å–ª–æ–≤–∏—è.

–ü—Ä–∏–º–µ—Ä —Å –ª–æ–≥–∏—á–µ—Å–∫–∏–º–∏ –æ–ø–µ—Ä–∞—Ç–æ—Ä–∞–º–∏:

–°–æ–≤–µ—Ç—ã –¥–ª—è —Ä–∞–±–æ—Ç—ã —Å —É—Å–ª–æ–≤–Ω—ã–º–∏ –æ–ø–µ—Ä–∞—Ç–æ—Ä–∞–º–∏

–ê–∫–∫—É—Ä–∞—Ç–Ω–æ–µ —Ñ–æ—Ä–º–∞—Ç–∏—Ä–æ–≤–∞–Ω–∏–µ:

–ü–æ–Ω—è—Ç–Ω—ã–µ —É—Å–ª–æ–≤–∏—è:

–ü–∏—à–∏—Ç–µ —É—Å–ª–æ–≤–∏—è, –∫–æ—Ç–æ—Ä—ã–µ –ª–µ–≥–∫–æ –ø–æ–Ω—è—Ç—å. –ï—Å–ª–∏ –æ–Ω–∏ —Å—Ç–∞–Ω–æ–≤—è—Ç—Å—è —Å–ª–∏—à–∫–æ–º —Å–ª–æ–∂–Ω—ã–º–∏, –ø–æ–¥—É–º–∞–π—Ç–µ –æ–± –∏—Ö —É–ø—Ä–æ—â–µ–Ω–∏–∏.

–†–∞–∑–º–µ—â–∞–π—Ç–µ —É—Å–ª–æ–≤–∏—è —Å–ª–µ–≤–∞ –Ω–∞–ø—Ä–∞–≤–æ –ø–æ —É–±—ã–≤–∞—é—â–µ–π –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç–∏ –∏—Ö –∏—Å—Ç–∏–Ω–Ω–æ—Å—Ç–∏, —á—Ç–æ–±—ã –±—ã—Ç—å —É–≤–µ—Ä–µ–Ω–Ω—ã–º, —á—Ç–æ —Å—Ä–∞–≤–Ω–µ–Ω–∏—è –≤—ã–ø–æ–ª–Ω—è—é—Ç—Å—è –≤ —Ç–æ–º –ø–æ—Ä—è–¥–∫–µ, –≤ –∫–æ—Ç–æ—Ä–æ–º –Ω—É–∂–Ω–æ.

–£—Å–ª–æ–≤–Ω—ã–µ –æ–ø–µ—Ä–∞—Ç–æ—Ä—ã –ø—Ä–µ–≤—Ä–∞—â–∞—é—Ç –≤–∞—à –∫–æ–¥ –≤ –±–æ–ª–µ–µ —É–º–Ω—ã–π –∏ –∞–¥–∞–ø—Ç–∏–≤–Ω—ã–π, –ø–æ–∑–≤–æ–ª—è—è –≤—ã–ø–æ–ª–Ω—è—Ç—å —Ä–∞–∑–ª–∏—á–Ω—ã–µ –¥–µ–π—Å—Ç–≤–∏—è –≤ –∑–∞–≤–∏—Å–∏–º–æ—Å—Ç–∏ –æ—Ç –≤—Ö–æ–¥–Ω—ã—Ö –¥–∞–Ω–Ω—ã—Ö –∏ —Ç–µ–∫—É—â–µ–≥–æ —Å–æ—Å—Ç–æ—è–Ω–∏—è –ø—Ä–æ–≥—Ä–∞–º–º—ã. –≠—Ç–æ ‚Äî –æ—Å–Ω–æ–≤–∞ —Å–æ–∑–¥–∞–Ω–∏—è –¥–∏–Ω–∞–º–∏—á–µ—Å–∫–∏—Ö –ø—Ä–æ–≥—Ä–∞–º–º.

‚Äã
## –°–ª–æ–≤–∞—Ä–∏

–°–ª–æ–≤–∞—Ä–∏ –≤ Python ‚Äî —ç—Ç–æ —É–¥–æ–±–Ω–∞—è —Å—Ç—Ä—É–∫—Ç—É—Ä–∞ –¥–∞–Ω–Ω—ã—Ö, –∫–æ—Ç–æ—Ä–∞—è –ø–æ–∑–≤–æ–ª—è–µ—Ç —Ö—Ä–∞–Ω–∏—Ç—å –∏ —É–ø—Ä–∞–≤–ª—è—Ç—å –¥–∞–Ω–Ω—ã–º–∏ –≤ –≤–∏–¥–µ –ø–∞—Ä "–∫–ª—é—á-–∑–Ω–∞—á–µ–Ω–∏–µ". –û–Ω–∏ –ø–æ—Ö–æ–∂–∏ –Ω–∞ –∞–¥—Ä–µ—Å–Ω—ã–µ –∫–Ω–∏–≥–∏, –≥–¥–µ –∫–∞–∂–¥–æ–º—É –∏–º–µ–Ω–∏ (–∫–ª—é—á—É) —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É–µ—Ç –Ω–æ–º–µ—Ä —Ç–µ–ª–µ—Ñ–æ–Ω–∞ (–∑–Ω–∞—á–µ–Ω–∏–µ). –í–æ—Ç —á—Ç–æ –Ω—É–∂–Ω–æ –∑–Ω–∞—Ç—å –Ω–æ–≤–∏—á–∫—É –æ —Å–ª–æ–≤–∞—Ä—è—Ö —Å –ø—Ä–æ—Å—Ç—ã–º–∏ –æ–±—ä—è—Å–Ω–µ–Ω–∏—è–º–∏ –∏ –ø—Ä–∏–º–µ—Ä–∞–º–∏:
–ß—Ç–æ —Ç–∞–∫–æ–µ —Å–ª–æ–≤–∞—Ä—å?

- –°–ª–æ–≤–∞—Ä—å ‚Äî —ç—Ç–æ –Ω–µ—É–ø–æ—Ä—è–¥–æ—á–µ–Ω–Ω–∞—è –∫–æ–ª–ª–µ–∫—Ü–∏—è –ø–∞—Ä "–∫–ª—é—á-–∑–Ω–∞—á–µ–Ω–∏–µ".
- –ö–ª—é—á–∏ –¥–æ–ª–∂–Ω—ã –±—ã—Ç—å —É–Ω–∏–∫–∞–ª—å–Ω—ã–º–∏ –∏ –Ω–µ–∏–∑–º–µ–Ω—è–µ–º—ã–º–∏ –¥–∞–Ω–Ω—ã–º–∏ (–Ω–∞–ø—Ä–∏–º–µ—Ä, —Å—Ç—Ä–æ–∫–∏, —á–∏—Å–ª–∞, –∫–æ—Ä—Ç–µ–∂–∏).
- –ó–Ω–∞—á–µ–Ω–∏—è –º–æ–≥—É—Ç –±—ã—Ç—å –ª—é–±–æ–≥–æ —Ç–∏–ø–∞ –¥–∞–Ω–Ω—ã—Ö –∏ –º–æ–≥—É—Ç –¥—É–±–ª–∏—Ä–æ–≤–∞—Ç—å—Å—è.
–°–æ–∑–¥–∞–Ω–∏–µ —Å–ª–æ–≤–∞—Ä—è

–ú–æ–∂–Ω–æ —Å–æ–∑–¥–∞—Ç—å –ø—É—Å—Ç–æ–π —Å–ª–æ–≤–∞—Ä—å –∏–ª–∏ —Å—Ä–∞–∑—É –∑–∞–ø–æ–ª–Ω–∏—Ç—å –µ–≥–æ –∑–Ω–∞—á–µ–Ω–∏—è–º–∏:

```python

# –°–ª–æ–≤–∞—Ä—å —Å –Ω–∞—á–∞–ª—å–Ω—ã–º–∏ –∑–Ω–∞—á–µ–Ω–∏—è–º–∏
```

–ò–∑–º–µ–Ω–µ–Ω–∏–µ –∏ –¥–æ–±–∞–≤–ª–µ–Ω–∏–µ —ç–ª–µ–º–µ–Ω—Ç–æ–≤

–í—ã –º–æ–∂–µ—Ç–µ –¥–æ–±–∞–≤–ª—è—Ç—å –Ω–æ–≤—ã–µ –ø–∞—Ä—ã "–∫–ª—é—á-–∑–Ω–∞—á–µ–Ω–∏–µ" –∏–ª–∏ –∏–∑–º–µ–Ω—è—Ç—å –∑–Ω–∞—á–µ–Ω–∏—è —Å—É—â–µ—Å—Ç–≤—É—é—â–∏—Ö –∫–ª—é—á–µ–π.

–£–¥–∞–ª–µ–Ω–∏–µ —ç–ª–µ–º–µ–Ω—Ç–æ–≤

–ü—Ä–∏–º–µ—Ä—ã –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è

–°–ª–æ–≤–∞—Ä–∏ –Ω–µ–∑–∞–º–µ–Ω–∏–º—ã, –∫–æ–≥–¥–∞ –Ω—É–∂–Ω–æ –±—ã—Å—Ç—Ä–æ –∏—Å–∫–∞—Ç—å –¥–∞–Ω–Ω—ã–µ –ø–æ –∫–ª—é—á—É. –ù–∞–ø—Ä–∏–º–µ—Ä, –æ–Ω–∏ —É–¥–æ–±–Ω—ã –¥–ª—è —Ö—Ä–∞–Ω–µ–Ω–∏—è –¥–∞–Ω–Ω—ã—Ö –∫–æ–Ω—Ñ–∏–≥—É—Ä–∞—Ü–∏–∏, –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–∏—è –∞—Å—Å–æ—Ü–∏–∞—Ü–∏–π –∏–ª–∏ —Ä–∞–±–æ—Ç—ã —Å JSON-–¥–∞–Ω–Ω—ã–º–∏, –≥–¥–µ –∫–ª—é—á–∏ –∏ –∑–Ω–∞—á–µ–Ω–∏—è —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É—é—Ç —Å—Ç—Ä—É–∫—Ç—É—Ä–µ –¥–∞–Ω–Ω—ã—Ö.

–ù–∏–∂–µ –ø—Ä–∏–≤–µ–¥–µ–Ω –ø—Ä–∏–º–µ—Ä –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è —Å–ª–æ–≤–∞—Ä—è:
